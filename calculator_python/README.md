#Python Calculator 

## Overveiw
This project is the basic calculator programmed in python which ask user 2 number to perform calculation on and then ask for operation and perform the calculation. It can perform addition, substraction, multiplication and division based on what user entered.

## How to run 
1. Open your terminal or command prompt.
2. Navigate to the'calculator_python' folder.
3. Run the following command:
   'calculator.py'
