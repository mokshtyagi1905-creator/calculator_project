# ⚙️ C Calculator

A simple calculator program written in C to practice basic programming concepts.

## ✨ Features

- Addition
- Subtraction
- Multiplication
- Division

## 🛠️ Concepts Used

- Variables
- Input/Output (`scanf()`, `printf()`)
- `switch` statement
- `if` statement

## 🔮 Future Improvements

1. Add a menu-based interface
2. Handle invalid input
3. Add more mathematical operations
4. Allow multiple calculations without restarting
5. Save calculation history using file handling
6. Create a graphical user interface (GUI)

## 🎯 Purpose

This project is part of my C programming learning journey. It helped me practice user input, decision-making using `switch`, and basic arithmetic operations.
